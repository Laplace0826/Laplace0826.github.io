我的快速入口首頁

讓常用網站更好看、 更快打開

這是一個適合直接部署到 GitHub Pages 的純 HTML 首頁。使用分類卡片、icon 圖示、玻璃擬態與漸層視覺，打造現代、有設計感的快捷入口頁面。

小提示：你可以直接修改每個 <a> 連結的名稱、網址與說明，然後把這個檔案命名為 index.html，上傳到 GitHub Pages 就能直接使用。
